API Endpoints for E-Commerce

1. CRUD for the administrator

Products

  • POST /admin/products - Create a new product
  • GET /admin/products - Get all products
  • GET /admin/products/:id - Get product by ID
  • PUT /admin/products/:id - Update a product
  • DELETE /admin/products/:id - Delete a product

Categories

  • POST /admin/categories - Create a new categories
  • GET /admin/categories - Get all categories
  • PUT /admin/categories/:id - Update a specific category
  • DELETE /admin/categories/:id - Delete a specific category

Orders

  • GET /admin/orders - Create all orders
  • GET /admin/orders - Get a specific order
  • PUT /admin/orders/:id - Update a specific order (e.g. change status)
  • DELETE /admin/orders/:id - Delete a specific order (e.g. cancel an order)

Users

  • GET /admin/users - Create all user
  • GET /admin/users - Get a specific user
  • DELETE /admin/users/:id - Delete a specific user

2. CRUD for the user

Products and navigation

  • GET /products - Get all products
  • GET /products:id - Get all categories

Cart

  • GET /cart - Get all products in the cart
  • POST /cart - Add a product to the cart
  • PUT /cart/:id - Update the quantity of a specific product in the cart
  • DELETE /cart/:id - Remove a specific product from the cart

Orders

  • POST /orders - Create a new order
  • GET /orders - Get all orders
  • GET /orders/:id - Get a specific order

Authentication

  • POST /auth/register - Register a new user
  • POST auth/login - Login a user
  • POST auth/logout - Logout a user
  • POST auth/refresh - Refresh the access token

Profile

  • GET /profile - Get the user's profile
  • PUT /profile - Update the user's profile
